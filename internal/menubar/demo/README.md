# Demo avatars

Original vector art for `theirtime menubar --demo` (screenshots and README hero).

| File | Character | Theme |
|------|-----------|-------|
| `bob.svg` / `bob.png` | Otter with glasses | West-coast demo teammate |
| `ann.svg` / `ann.png` | Owl with star | East-coast demo teammate |

**License:** Same as the theirtime project ([MIT](../../../LICENSE)). You may use, modify, and redistribute these assets with the project.

PNG files are rasterized from the SVG sources for menu bar embedding (`128×128` via `sips`). Regenerate:

```bash
./scripts/rasterize-demo-avatars.sh
```

Palette matches [brand assets](../../../assets/brand/README.md): teal `#0D9488`, amber `#F4A23B`, charcoal `#3C3C38`.

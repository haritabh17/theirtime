# theirtime brand assets

Logo for **theirtime** — your coworkers' local time, right in your menu bar.

## Files
- `icon/theirtime-icon.svg` — app icon (teal), scalable. `-dark.svg` is the dark-mode variant.
- `icon/png/` — rasterized icons, 16-1024 px (plus dark 512/1024).
- `icon/theirtime.icns` — macOS icon bundle.
- `wordmark/` — the "theirtime" wordmark, **outlined to vector** (no font needed). light + dark.
- `lockup/` — icon + wordmark horizontal logo, SVG and PNG, light + dark.
- `menubar/` — monochrome glyph for the macOS status bar (a template image; the OS tints it).

## Use light vs dark
Light assets on light backgrounds; dark assets on dark backgrounds. The wordmark's "their" flips
between charcoal and off-white; "time" stays teal in both.

## Palette
- Teal (primary)      #0D9488
- Teal (accent)       #14B8A6
- Teal (dark mode)    #2DD4BF
- Dark icon ground    #0B3F3A
- Amber (their hand)  #F4A23B
- Charcoal (their)    #3C3C38
- Off-white (their)   #E8EAED

## Type
Wordmark set in **Baloo 2 SemiBold** (Open Font License). It is outlined in the SVGs, so the font
does not need to be installed to render the logo.
